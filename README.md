# My Survival RPG

This is an advanced React-based survival RPG game featuring multiple game mechanics such as resource management, village building, combat encounters, tech tree upgrades, and more.

## Features

- **Survival Mechanics:** Hunger, thirst, energy, and health.
- **Village Management:** Build farms, mines, workshops, and recruit villagers.
- **Skill Progression:** Improve gathering, crafting, combat, and exploration skills.
- **Random Encounters:** Fight raiders and wild animals or trade with wandering traders.
- **Tech Tree & Tasks:** Unlock technologies and complete tasks to earn research points.
- **Dynamic Environment:** Day/night cycle, changing weather, and temperature effects.
